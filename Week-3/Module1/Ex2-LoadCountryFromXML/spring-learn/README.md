# Exercise 2 - Spring Core - Load Country from Spring Configuration XML

## Description

Loading country bean from Spring XML configuration file using ClassPathXmlApplicationContext.

## Technologies
- Java
- Spring Framework
- XML Configuration
- SLF4J Logging
- Maven

## Implementation Details
- Load bean from `country.xml` using `ClassPathXmlApplicationContext`
- Retrieve country bean by id
- Display country details with logging
