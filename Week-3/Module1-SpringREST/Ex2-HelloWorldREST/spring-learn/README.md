# Exercise 2 - Hello World RESTful Web Service

## 📌 Objective

Develop a simple RESTful Web Service using Spring Boot that returns "Hello World!!".

## 🚀 Features

- Spring Boot REST API
- GET Endpoint
- SLF4J Logging
- Browser Testing
- Postman Testing

## 🛠 Technologies

- Java 17
- Spring Boot
- Spring Web
- Maven

## 🌐 Endpoint

GET /hello

Response:

Hello World!!

## ▶️ Run

```bash
mvn spring-boot:run
