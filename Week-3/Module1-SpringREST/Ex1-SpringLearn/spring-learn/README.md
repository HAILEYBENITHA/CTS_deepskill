# Exercise 1 - Spring Boot Project Setup

## 📌 Objective

Create a Spring Boot Web project using Maven and verify application startup with logging.

## 🚀 Features

- Spring Boot Web Project
- Spring Boot DevTools
- Maven Build
- Application Logging
- Embedded Tomcat

## 🛠 Technologies

- Java 17
- Spring Boot
- Spring Web
- Maven

## 📂 Project Structure

```
src
├── main
│   ├── java
│   └── resources
├── test
└── pom.xml
```

## ▶️ Build

```bash
mvn clean package
```

## ▶️ Run

```bash
mvn spring-boot:run
```
