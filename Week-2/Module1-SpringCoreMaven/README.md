# Module 1 - Spring Core and Maven

## Description

This module introduces the core concepts of the Spring Framework and Maven. The exercises demonstrate dependency injection, inversion of control, project configuration, and Spring Boot application development.

---

## Exercises

### Exercise 1
Configuring a Basic Spring Application

### Exercise 2
Implementing Dependency Injection

### Exercise 4
Creating and Configuring a Maven Project

### Exercise 5
Configuring the Spring IoC Container

### Exercise 7
Constructor and Setter Injection

### Exercise 9
Creating a Spring Boot Application

---

## Concepts Covered

- Spring Core
- Dependency Injection
- Inversion of Control (IoC)
- Bean Configuration
- Constructor Injection
- Setter Injection
- Maven Build Tool
- Spring Boot

Successfully developed Spring applications using dependency injection, IoC container configuration, Maven, and Spring Boot.
