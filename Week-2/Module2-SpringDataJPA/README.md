# Module 2 - Spring Data JPA

## Description

This module demonstrates database operations using Spring Data JPA and Hibernate. The exercises cover CRUD operations, query methods, object-relational mapping, HQL, and native SQL queries.

---

## Exercises

### Exercise 1
Spring Data JPA Quick Example

- Difference between JPA, Hibernate and Spring Data JPA
- Implement Country Service
- Find Country by Country Code
- Add New Country

### Exercise 2
Query Methods

- Implement Spring Data JPA Query Methods

### Exercise 3
ORM Mapping, HQL and Native Query

- Object Relational Mapping
- Hibernate Query Language (HQL)
- Native SQL Query
