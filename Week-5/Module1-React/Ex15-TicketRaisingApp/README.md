# Ex15 - Ticket Raising App (ticketraisingapp) - ReactJS HOL

Source: ReactJS-HOL.docx (Digital Nurture Java FSE - Deepskilling / React)

## Objectives
- React forms
- Controlled components
- Various input controls (textbox, textarea, button)
- Handling and submitting forms

## Task
Create a React App named **ticketraisingapp** to raise a complaint and get it
resolved. Create a component named **ComplaintRegister** with a form containing:
- a **textbox** to enter the employee name
- a **textarea** to enter the complaint

Use the **`handleSubmit`** event of the button to submit the complaint and
generate a **Reference number** for further follow-ups in an alert box.

## Concepts Covered
- Controlled components (state-bound inputs)
- `onChange` handlers for `input` and `textarea`
- `onSubmit` form handling with `event.preventDefault()`
- Generating a reference number on submit

## Run
```
npm start
```
