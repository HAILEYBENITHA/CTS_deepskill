# Ex13 - Blogger App (bloggerapp) - ReactJS HOL

Source: ReactJS-HOL.docx (Digital Nurture Java FSE - Deepskilling / React)

## Objectives
- Various ways of conditional rendering
- Render multiple components
- List components
- Keys in React
- Extracting components with keys
- `React.map()` / `map()` function

## Task
Create a React App named **bloggerapp** with **3 components**:
1. **Book Details**
2. **Blog Details**
3. **Course Details**

Implement this with as many ways as possible of **conditional rendering**.

## Implementation notes
- `BookDetails`, `BlogDetails`, `CourseDetails` each render a list using the
  `map()` function with `key` props.
- Conditional rendering techniques demonstrated in `App.js`:
  - **if / else with element variables**
  - **logical && operator**
  - **ternary operator**

## Concepts Covered
- Conditional rendering (if/else, &&, ternary)
- Multiple component rendering
- List components with `map()`
- Keys for list items

## Run
```
npm start
```
