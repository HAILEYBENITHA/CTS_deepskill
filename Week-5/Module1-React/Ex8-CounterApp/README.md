# Ex8 - Counter App (counterapp)

## Objective
Create a React app named **counterapp** with a component named **CountPeople**
that tracks the number of people who enter and exit a mall.

## Requirements
- Two methods:
  - `UpdateEntry()` - displays / increments the number of people who entered.
  - `UpdateExit()` - displays / increments the number of people who exited.
- Use a **constructor** and **state** to store `entrycount` and `exitcount`.
- Two buttons:
  1. **Login** - when clicked, `entrycount` increments by 1.
  2. **Exit** - when clicked, `exitcount` increments by 1.

## Concepts Covered
- Class component
- Constructor and state
- Event handling (`onClick`)
- `setState`

## Output
The page shows the current entry and exit counts, updating live as the Login
and Exit buttons are clicked.
