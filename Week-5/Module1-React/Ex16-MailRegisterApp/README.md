# Ex16 - Mail Register App (mailregisterapp) - ReactJS HOL

Source: ReactJS-HOL.docx (Digital Nurture Java FSE - Deepskilling / React)

## Objectives
- React Forms validation
- Differences between React Form and HTML Form
- Controlled components
- React Form input controls
- Handling and submitting React Forms

## Task
Create a React App named **mailregisterapp** with a component named
**Register.js**. Create a form that accepts **name**, **email** and
**password** and validates the fields:

| Field | Validation |
|-------|------------|
| Name | At least 5 characters |
| Email | Must contain `@` and `.` |
| Password | At least 8 characters |

Implement validations through the **event handler** (`onChange`) and the
**event submit** (`onSubmit`) of the form.

## Concepts Covered
- Controlled components
- Field-level validation on `onChange`
- Whole-form validation on `onSubmit`
- Displaying validation errors

## Run
```
npm start
```
