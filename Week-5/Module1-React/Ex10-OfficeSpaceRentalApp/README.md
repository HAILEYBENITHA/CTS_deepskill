# Ex10 - Office Space Rental App (officespacerentalapp) - ReactJS HOL

Source: ReactJS-HOL.docx (Digital Nurture Java FSE - Deepskilling / React)

## Objectives
- Use JSX syntax in React applications
- Use inline CSS in JSX
- Create React nodes with JSX
- Render JSX to the DOM
- Use JavaScript expressions in JSX

## Task
Create a React Application named **officespacerentalapp** which uses React JSX
to create elements, attributes and render the DOM to display the page.

1. Create an element to display the **heading** of the page.
2. Add an **attribute** to display the **image** of the office space.
3. Create an **object** of office to display details like **Name**, **Rent**
   and **Address**.
4. Create a **list of objects** and loop through the office space items to
   display more data.
5. Apply CSS: display the **Rent in Red if below 60000** and **Green if above
   60000** (inline CSS in JSX).

## Concepts Covered
- JSX element creation
- JSX attributes (e.g., `src`, `alt`)
- JavaScript expressions inside JSX `{}`
- Inline CSS via the `style` prop
- Conditional styling with a ternary expression
- `.map()` to loop over a list of objects

## Run
```
npm start
```
