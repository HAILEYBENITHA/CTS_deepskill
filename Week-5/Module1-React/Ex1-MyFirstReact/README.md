# Ex1 - My First React (myfirstreact)

## Objective
Create a new React application named **myfirstreact** and run it to print
**"welcome to the first session of React"** as the heading of the page.

## Steps
1. Install Node.js and npm from https://nodejs.org/en/download/
2. Install create-react-app:
   ```
   npm install -g create-react-app
   ```
3. Create the React application:
   ```
   npx create-react-app myfirstreact
   ```
4. Navigate into the project folder:
   ```
   cd myfirstreact
   ```
5. Open the folder in Visual Studio Code.
6. Open `src/App.js`.
7. Remove the current content of `App.js`.
8. Replace it with a component that renders the heading
   `welcome to the first session of React`.
9. Run the application:
   ```
   npm start
   ```
10. Open a browser and go to `http://localhost:3000`.

## Output
The page displays the heading:

> welcome to the first session of React
