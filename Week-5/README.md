# Week 5 - CTS Deep Skill Training

## Overview

This repository contains the exercises completed during Week 5 of the Cognizant Deep Skill Program.

The exercises focus on front-end development using **React** and **Angular**, covering JSX, components, props & state, event handling, conditional rendering, forms, validation, React Router, Context API, REST API consumption, and unit testing with Jest & Enzyme.
