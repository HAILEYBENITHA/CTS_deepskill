# Week 1 - CTS Deep Skill Training

## Overview

This repository contains the exercises completed during Week 1 of the Cognizant Deep Skill Program.

The exercises focus on Java Design Patterns, Data Structures & Algorithms, PL/SQL Programming, Unit Testing using JUnit & Mockito, and Logging using SLF4J.
