# Module 1 - Design Patterns & Principles

## Description

This module introduces commonly used object-oriented design patterns.

The exercises demonstrate how design patterns improve code reusability, flexibility and maintainability.

---

## Exercises

### Exercise 1
Singleton Pattern

Objective:
Implement the Singleton Design Pattern to ensure only one object is created.

---

### Exercise 2
Factory Method Pattern

Objective:
Implement the Factory Method Design Pattern to create objects without exposing object creation logic.

---

## Concepts Covered

- Singleton Pattern
- Factory Method Pattern
- Object-Oriented Design
- Encapsulation
- Loose Coupling

---

## Technologies

- Java
